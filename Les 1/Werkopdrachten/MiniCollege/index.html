OOP JavaScript

What is JavaScript?

"JavaScript is an object-oriented scripting language designed primarily for adding interactivity to Web pages and creating Web applications."

  • Object-oriented language with prototypal inheritance
  • Built-in objects and custom objects
  • No classical inheritance, but can be emulated
  • Objects inherit properties directly from each other, forming the prototype chain

OOP JavaScript

Why use OOP?

  • Not lineair script, but reusable chunks of code
  • Structured, clear and often shorter code
  • Makes it easy to extend and customize code
  • No abundance of global stuff
  • It's easier and more fun

Best practices & coding conventions

For starters

  • Don't use global variables/objects
  • Use short clear (meaningful) names
  • Constructor function start with capital
  • Constants with all captials
  • camelCase the rest
  • Place external scripts at the bottom of the page
  • Comment your code (Docco)
