21. Ontwerp de interface
  1. Maak een screenflow van de verschillende states van je applicatie.
  2. Zoeken voorbeelden van elementen / interacties die je wil toepassen in je interface
  3. Teken deze interacties in je screenflow
22. Flexbox

Pas flexbox (gedeeltelijk) toe om de layout van je web app vorm te geven.

A Complete Guide to Flexbox
23. Functional animation

Voeg met behulp van een css transition een functionele animatie toe aan je web app. Een functionele animatie is bijvoorbeeld het in beeld. Gebruik alleen css properties die met Hardware Acceleration worden gerenderd.

High performance animations
24. Gestures

Voeg met behulp van een of meer touch event(s) gestures toe aan je interface. Je mag vanilla JavaScript gebruiken, maar ook een library zoals bijvoorbeeld Hammer.js

Hammer.js
