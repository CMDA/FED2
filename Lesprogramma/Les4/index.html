15.2 Haal JSON data op van de movie API en parse de data in je HTML

Gebruik het xhr object om de data van de API op te halen. De API vind je hier: http://dennistel.nl/movies

Gebruik JSON.parse(response) om de data om te zetten van een string naar JavaScript objecten, en laad de data in je HTML via Transparency

16. LocalStorage

Sla de data die je ophaalt van de API op in LocalStorage en check voordat je de API aanspreekt of de data reeds in LocalStorage bestaat.

Gebruik dit artikel als referentie: How to use LocalStorage for JavaScript

17. Application cache (optioneel)

Gebruik HTML5 Application Cache om static files (css, javascript, etcetera) in de browser cache te kunnen opslaan en bezoekers bij terugkomst op de website de mogelijkheid te geven de website te bekijken zonder toegang tot internet

Busting cache with Grunt
18. Web Workers (optioneel)

Maak een webworker aan die de data om de 5 seconde ophaalt van de Movie API en doorstuurt naar de web app. In de web app overschrijf je de data in localStorage.

Gebruik dit artikel als referentie: Using Web Workers to speed up your JavaScript Application
