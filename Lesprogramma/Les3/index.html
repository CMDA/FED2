10. Basic Skeleton

We stappen af van het GEO script en gaan beginnen met de deeltoets en gaan toewerken naar de eindopdracht. In je GitHub repository ga je nu werken in de WebApp folder.

Volg deze stappen om je basic skeleton neer te zetten

  • Neem de HTML over vanaf: voorbeel HTML, en zet dit in je eigen index.html
  • Voeg onderstaande objecten toe aan je app.js:
    1. NameSpace of Module pattern
    2. Self Invoking Anonymous Function
    3. 'Controller' object (starting point voor je applicatie)
11. Router
  • Bestudeer Routie.js: http://projects.jga.me/routie/
  • Download het script in de vendor folder en voeg het toe aan 'index.html'
  • Maak in 'app.js' een nieuw 'router'-object aan met een 'init'-methode
  • Voeg aan de 'init'-methode de 'routes' toe voor de 'about' en 'movies' sections
  • Laat voor beide routes in de console zien welke route aangeklikt is
  • Roep in de 'init'-methode van het 'controller'-object, de 'init'-methode van het 'router'-object aan
12. Templates

12.1: Data object

  • Maak een 'content'-object aan met daarin een 'about'-object en een 'movies'-array.
  • Het 'about'-object bevat twee properties; 'titel' en 'description'
  • De 'movies'-array bevat voor elke film een object met de properties; 'title', 'releaseDate', 'description' en 'cover'

12.2: Transparancy template engine

  • Bestudeer Transparency.js: https://github.com/leonidas/transparency
  • Download het script in de vendor folder en voeg het toe aan 'index.html'
  • Maak in 'app.js' een nieuw 'sections'-object aan met een 'init'-methode, een 'about'-methode, een 'movies'-methode
  • Haal alle data uit de html en laat alleen de structuur staan.
  • Render in de 'about'-methode en 'movies'-methode met Transparency de corresponderende data uit oefening 12.1 in de juiste sectie van de pagina.
  • Roep in de 'init'-methode van het 'sections'-object de 'about'-methode en 'movies'-methode aan
  • Roep in de 'init'-methode van het 'controller'-object, de 'init'-methode van het 'sections'-object aan
13. Toggle sections

Het wordt tijd om de router en de sections met elkaar samen te laten werken!

  • Geef in je stylesheet alle sections een display:none, en alle sections met een class 'active' display:block
  • Voeg een 'toggle'-methode toe aan het 'sections'-object. Deze methode heeft een parameter 'section' en regelt welke section gedisplayed wordt
  • Roep vanuit het 'router'-object de zojuist aangemaakte 'toggle'-methode aan met de juiste parameter (corresponderend met de gekozen section)
  • De 'toggle'-methode regelt welke section een class active krijgt
  • Bekijk hier: How to forget about jquery and start using native hoe je met native JavaScript elementen kan selecteren en hoe je class attributen kan toevoegen/verwijderen/toggelen
14. Best practices

Voeg aan het README.md bestand van je repository minimaal 5 best practices toe met betrekking tot JavaScript (zoek op via Google en overleg met je duo). ALs je tijd over hebt kan je ook CSS en HTML best practices toevoegen.
